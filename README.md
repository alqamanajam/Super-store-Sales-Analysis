# 🛒 Supermarket Sales Analysis & Customer Insights

## 📌 Project Overview
Analysed 541,910 retail transactions from a UK-based online supermarket 
to uncover revenue trends, top products, country performance and 
customer purchasing behaviour.

**Role:** Junior Data Analyst  
**Dataset:** Kaggle — Online Retail Transactions  
**Period:** December 2010 – December 2011  

---

## 🛠️ Tools Used
| Tool | Purpose |
|------|---------|
| Python (Pandas, Matplotlib, Seaborn) | Data cleaning & visualisation |
| Power BI (DAX, Power Query) | Interactive dashboard |

---

## 📊 Key Findings
- 💰 Total Revenue: **£10.6 Million** across 13 months
- 🧾 Total Orders: **19,960**
- 👥 Total Customers: **4,338** across 38 countries
- 🏆 Best Month: **November 2011 — £1.5M** (holiday season peak)
- 🇬🇧 UK contributes **84.6%** of all revenue
- 📦 Top Product: **Paper Craft Little Birdie** — 80,995 units sold
- 📅 Busiest Day: **Thursday**

---

## 🔍 Data Quality Issues Found
| Issue | Records |
|-------|---------|
| Missing Customer IDs | 135,080 (24.9%) |
| Missing Descriptions | 1,454 |
| Negative Quantities (Returns) | 10,624 |
| Invalid Prices | Removed |
| **Final Clean Dataset** | **530,105 rows** |
